# Operator

營運企劃設計問卷，做市場調查，設定受眾TA(Target Audience)  
尋找市面上優秀的作品，遊玩之後寫競品分析報告,同時也要分析遊戲核心玩法、核心機制  

這個職位是吸收了商業中PM(Product Manager)的概念  
重點在於做出TA喜歡，賣錢的產品  
營運企劃可能不算在開發部門而是隸屬商務部門下

Causual Game會需要強大的營運企劃，需要知道市場喜歡甚麼，能看數據改進遊戲玩法
